# movies-search

Movies Search App that use: 
 * New versions of Javascript (es6, ...)
 * Webpack Babel configuration that support
 * Working with DOM using js.

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm run start
```

### Build
Compile and build

```sh
$ npm run build
```
